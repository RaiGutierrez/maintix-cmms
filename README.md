# Maintix CMMS

Maintix is a web-based CMMS (Computerized Maintenance Management System) designed for industrial environments.

It allows companies to manage machines, work orders, maintenance tasks and operational visibility in a simple interface.

## Features

• Machine registry  
• Work order management  
• Maintenance status tracking  
• Dashboard with operational insights  
• Priority and status indicators
